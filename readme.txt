== Description ==

Wp J Facebook Likebox is slide up jquery window to represent facebook like box with customize feature. This plugin has both php and short code option. Please use [wpjflb] or wpjflb_loadpagefnc(); in your site for calling the plugin.

== Installation ==

1. Upload 'wpjfacebook_likebox.php','adminpage.php' and lib to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings => Wp J Facebook Likebox and configure the plugin according to your need
1. Place [wpjflb] on post or page or  `<?php wpjflb_loadpagefnc(); ?>` in your templates
